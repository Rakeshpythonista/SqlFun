What is PL/SQL?
PL/SQL stands for Procedural Language Extension to SQL.

PL/SQL Blocks
  >DECLARE(Optional)
  >BEGIN(Optional)
  >EXCEPTION(Optional)
  >END;(Optional)

Three types of blocks
  >AnonymousBlocks
  >Procedures
  >Functions
---------------------------------------------------
Let's start coding
PL/SQL outputs
  > Not an output language.
  > SET SERVEROUTPUT ON;
  > DBMS_OUTPUT.PUT_LINE(output_text);

Example 1:
BEGIN
    DBMS_OUTPUT.PUT_LINE('Hello World');
END;

>>>No output

Example 2:
SET SERVEROUTPUT ON;
BEGIN
    DBMS_OUTPUT.PUT_LINE('Hello World');
END;

>>>Hello World

Example 3:
SET SERVEROUTPUT ON;
DECLARE
    Messsage VARCHAR2(20) := 'Hello World';
BEGIN
    DBMS_OUTPUT.PUT_LINE(Messsage);
END;

>>>Hello World

Example 4:
SET SERVEROUTPUT ON;
BEGIN
    DBMS_OUTPUT.PUT_LINE('Hello World');
    BEGIN
        DBMS_OUTPUT.PUT_LINE('Hello Rakesh');
    END;
END;

>>>Hello World
   Hello Rakesh

Example 5:
SET SERVEROUTPUT ON;
DECLARE
    Messsage VARCHAR2(20) := 'Hello World';
BEGIN
    DBMS_OUTPUT.PUT_LINE(Messsage);
    BEGIN
        DBMS_OUTPUT.PUT_LINE('Hello Rakesh');
    END;
END;

>>>Hello World
   Hello Rakesh
